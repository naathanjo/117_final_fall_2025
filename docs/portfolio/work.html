Inventory Management System

Database Designer & Developer

Designed and implemented a relational database system for a local retail business to track inventory levels in real-time. Created normalized SQL schemas with proper indexing for optimal query performance. Developed automated low-stock alert notifications using Python scripts that email managers when inventory falls below threshold levels. The system reduced manual stock counts by 80% and improved order accuracy.
